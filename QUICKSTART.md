# 🚀 快速开始

## 第一次使用（3 分钟）

### 1️⃣ 安装插件

```bash
# 1. 打开 Chrome
# 2. 访问 chrome://extensions/
# 3. 开启"开发者模式"
# 4. 点击"加载已解压的扩展程序"
# 5. 选择 redbook-tools 文件夹
```

### 2️⃣ 配置内容

编辑 `source/mock.json`：

```json
{
  "title": "你的标题",
  "content": "你的正文内容",
  "images": ["图片URL1", "图片URL2"],
  "tags": ["标签1", "标签2"],
  "commodityId": ["商品ID1"]
}
```

### 3️⃣ 开始使用

```
1. 打开小红书发布页面
2. 点击插件图标
3. 点击"一键自动填充"
4. 等待 10-15 秒
5. 检查内容
6. 手动点击"发布"
```

## ❓ 遇到问题？

### 没有反应

1. 确认在小红书发布页面
2. F12 查看 Console 错误信息
3. 重新加载插件
4. 刷新页面重试

### 部分功能失败

- 查看 Console 日志
- 确认 mock.json 格式正确
- 手动补充失败的部分

### 需要帮助

- 查看 [GUIDE.md](./GUIDE.md) 详细指南
- 查看 [README.md](./README.md) 完整说明
- 提交 Issue
