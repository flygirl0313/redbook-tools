# Redbook Tools

小红书工具集浏览器插件，提供更好的使用体验。

## 功能特性

- 🚫 **隐藏广告** - 自动隐藏小红书页面中的广告内容
- 📥 **图片下载** - 一键下载小红书图片
- 📝 **文案复制** - 快速复制笔记文案内容

## 安装方法

### 开发模式安装
1. 克隆或下载本项目
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹

## 项目结构

```
redbook-tools/
├── manifest.json          # 插件配置文件
├── popup.html            # 弹窗页面
├── popup.css             # 弹窗样式
├── popup.js              # 弹窗逻辑
├── content.js            # 内容脚本
├── content.css           # 内容样式
├── background.js         # 后台脚本
├── icons/                # 图标目录
└── README.md             # 项目说明
```

## 使用说明

1. 安装插件后，在小红书页面点击插件图标
2. 在弹窗中开启需要的功能
3. 功能会自动应用到当前页面
